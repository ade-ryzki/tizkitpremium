## Getting Started
  
### Installation
  
1. Clone this repos (Tickitz)
```sh
git clone https://github.com/ade-ryzki/tizkitpremium.git
```
2. Go To Folder Repo
```sh
code .
```
3. Install Module
```sh
npm install
```
  
4. Start website
```sh
npm start
```
5. Website
```sh
tizkitpremium.vercel.app
```
